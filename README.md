# 💰 Tool Tách Tiền Tự Động

Một công cụ đơn giản giúp tách số tiền từ danh sách đơn hàng và xuất nhanh kết quả để phục vụ ghép đơn, đối soát hoặc nhập liệu.

![Demo](./images/demo.png)

## 🚀 Tính năng

- Tự động nhận diện số tiền từ danh sách đơn hàng.
- Loại bỏ ký tự tiền tệ (đ, vnđ, dấu chấm phân cách).
- Xuất danh sách số tiền theo từng dòng.
- Nút Copy kết quả chỉ với 1 lần nhấn.
- Giao diện tối (Dark Mode) hiện đại, dễ sử dụng.
- Hoạt động trực tiếp trên trình duyệt, không cần cài đặt.

---

## 📋 Ví dụ

### Dữ liệu đầu vào

```text
PB18060004926 — 604.390đ
PB18060027207 — 395.194đ
```

### Kết quả

```text
604
395
```

Tool phù hợp cho:

- Ghép đơn điện.
